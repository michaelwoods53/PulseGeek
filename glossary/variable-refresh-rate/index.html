Variable Refresh Rate

Display tech that changes refresh rate to match frame rate.

In detail

Variable Refresh Rate lets a display adjust its refresh to the game’s frame rate. This reduces tearing and stutter and improves smoothness with fluctuating FPS.

Synonyms

adaptive sync

Abbreviations

VRR
