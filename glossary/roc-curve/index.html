ROC Curve

A plot that shows the trade-off between true positive rate and false positive rate across thresholds. It helps compare models for detection tasks.

In detail

The ROC curve charts true positive rate against false positive rate for different decision thresholds. It visualizes how a model balances detection sensitivity and noise. Area under the curve (AUC) summarizes overall performance. For cybersecurity, ROC analysis helps pick thresholds that limit alert fatigue while still catching real threats, especially when combined with precision-recall figures.

Synonyms

receiver operating characteristic

Abbreviations

ROC
