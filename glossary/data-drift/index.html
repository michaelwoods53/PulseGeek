Data Drift

Changes in the input data distribution that can reduce model quality, such as new vendors, pricing, or formats in finance systems.

In detail

Data drift is a shift in the statistical properties of input data used by a model. In finance, it may appear as new invoice formats, unusual seasonal patterns, or changes in customer behavior. Drift can break assumptions made during training and lead to higher error rates. Teams monitor features and inputs, set alerts, and compare live data with training baselines. Fixes include retraining, feature engineering, and updating validation rules.

distribution shift
