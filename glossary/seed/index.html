Seed

A starting value that makes random-looking results repeatable.

In detail

A seed initializes a random or pseudorandom process. Using the same seed and algorithm produces the same output, enabling reproducible builds, synchronized worlds, and consistent testing.

Synonyms

random seed, initial seed

Related terms

  • Procedural Generation
  • Pseudorandom Number Generator
  • Deterministic Build
