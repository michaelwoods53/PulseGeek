Network Latency

The time it takes for data to travel between your device and the game server.

In detail

Network latency is the end-to-end delay measured as round-trip time between a client and server. In gaming it affects hit registration and responsiveness. Lower latency generally improves the feel of multiplayer actions.

Synonyms

lag, delay

Abbreviations

RTT

Related terms

  • Ping
  • Jitter
  • Server Tick Rate
