Feature Engineering

Feature engineering transforms raw data into useful inputs for models. It improves signal and model performance through selection, encoding, and aggregation.

In detail

Feature engineering is the process of creating and preparing model inputs from raw data. It includes handling missing values, encoding categories, scaling numbers, creating aggregates, and combining signals across sources. Strong features can greatly improve predictive power and stability. Many teams use feature stores to standardize and share approved features across models. Good documentation and testing reduce leakage and keep features consistent in training and production.

Synonyms

feature creation, variable transformation

Abbreviations

FE
