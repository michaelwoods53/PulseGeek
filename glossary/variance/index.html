Variance

A measure of how much loot outcomes fluctuate around the average.

In detail

Variance describes streakiness in loot outcomes. Tools like pity, floor chances, and weighted tables help control variance for a fairer experience.

Synonyms

outcome spread

Related terms

  • RNG
  • Pity Timer
  • Expected Value
