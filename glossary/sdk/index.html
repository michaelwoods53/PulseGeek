SDK

A toolkit provided to develop content or mods for a game.

In detail

An SDK includes tools, headers, documentation, and samples to build mods or content. It may expose APIs, scripting, and asset pipelines supported by the game.

Synonyms

software development kit

Abbreviations

SDK
