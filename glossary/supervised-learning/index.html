Supervised Learning

A machine learning approach that trains models using labeled examples. It is common in phishing and malware classification where known good and bad samples guide learning.

In detail

Supervised learning uses labeled input-output pairs to train models that map features to target classes or values. In cybersecurity, it powers tasks such as phishing detection and malware classification by learning from curated datasets with known malicious and benign samples. Popular algorithms include logistic regression, tree ensembles, and deep neural networks. Quality labels, balanced classes, and careful evaluation are crucial to avoid biased models that miss threats or overwhelm analysts with false alarms.
