Large Language Model

A generative model trained to predict and produce human-like text.

In detail

Large language models learn statistical patterns of language from large corpora. They enable chat interfaces, summarization, question answering, and code suggestions. Businesses use LLMs to automate support, draft documents, and accelerate analysis.
