Mod Manager

A tool to install, sort, enable, disable, and update mods.

In detail

A mod manager handles mod installation, profiles, load order, conflict detection, and updates. It simplifies managing large mod lists and helps avoid file clutter by using virtual file systems or staging folders.

Synonyms

manager
