Keys

Cryptographic values some systems use to decrypt content.

In detail

Certain consoles require decryption keys for firmware or game content. Emulators may need user-provided keys that are dumped from your own hardware to function legally.

Synonyms

system keys, decryption keys

Related terms

  • BIOS
