Oracle

An oracle supplies off-chain data to on-chain contracts, such as prices or weather. Secure oracle design prevents manipulation and minimizes trust in single data sources.

In detail

An oracle is a mechanism that provides external data to smart contracts. Since blockchains cannot directly fetch web data, contracts rely on oracles for information like asset prices, randomness, or shipment status. Robust oracle systems aggregate multiple sources, use cryptographic proofs, and define clear update rules to reduce manipulation. Designers choose between push and pull models, consider latency, and plan for failure modes. Because oracles bridge on-chain and off-chain worlds, they are critical components that can impact security, availability, and correctness of contract logic.

Synonyms

data oracle, price oracle
