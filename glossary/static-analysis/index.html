Static Analysis

Examining software artifacts without running them. Useful features include opcodes, imports, and strings for malware classification.

In detail

Static analysis reviews code or binaries without execution to extract features such as opcode frequencies, imported functions, strings, and structural metadata. It is fast and safe but can be fooled by obfuscation and packing. In malware classification, static features complement dynamic traces and help models generalize when runtime data is unavailable.

Synonyms

code analysis without execution
