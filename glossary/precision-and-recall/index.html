Precision and Recall

Two metrics that show how well a model avoids false alarms (precision) and how many real cases it catches (recall).

In detail

Precision and recall are paired metrics for classification tasks. Precision measures how many flagged items were truly correct, helping reduce false positives in anomaly or fraud alerts. Recall measures how many actual issues the model caught, focusing on sensitivity. Finance teams balance both using thresholds, cost-weighted metrics, and business rules. Monitoring precision and recall over time helps maintain control performance and user trust.

Synonyms

classification metrics
