True Positive Rate

Fraction of real threats the model catches.

In detail

True positive rate, or recall, measures the proportion of actual positives that are correctly identified. In cyber defense, high recall ensures fewer missed attacks.

Synonyms

Recall, Sensitivity

Abbreviations

TPR

Related terms

  • False Positive Rate
  • Precision-Recall Curve
  • Confusion Matrix
