Input Validation

Ensuring player inputs are possible and allowed.

In detail

Input validation checks movement, firing, and ability use against rules and physics. It rejects impossible actions, rate-limits spam, and helps prevent speed, teleport, or macro abuses.

Synonyms

server-side checks
