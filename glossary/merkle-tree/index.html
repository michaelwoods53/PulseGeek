Merkle Tree

A Merkle tree is a tree of hashes that summarizes many data items with a single root. It enables quick proofs that specific items are included without downloading all data.

In detail

A Merkle tree organizes data in pairs, hashing each pair up the tree until a single root hash remains. That root commits to all leaves beneath it. Anyone can verify a specific item is included by checking a short Merkle proof, which consists of sibling hashes along the path to the root. This saves bandwidth and storage because you do not need the full dataset to verify integrity. Blockchains use Merkle trees to summarize transactions in a block header and to allow light clients to verify inclusion. The concept supports efficient auditing, state synchronization, and fraud proofs in various scaling and verification schemes across distributed systems.
