Fraud Detection

The use of data and models to spot and stop suspicious transactions, chargebacks, and identity abuse before losses occur.

In detail

Fraud detection uses data signals and machine learning models to identify suspicious activity in payments, credit cards, refunds, and account access. Features can include velocity, device, location, merchant category, and behavior patterns. Teams measure performance with precision, recall, and cost-based metrics. Controls include automated holds, step-up verification, and case management. Ongoing monitoring is needed to combat evolving tactics and reduce both false positives and fraud losses.

Synonyms

fraud analytics, fraud monitoring

Abbreviations

FD
