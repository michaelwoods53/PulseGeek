Rate Limiting

Restricting the frequency of actions or requests.

In detail

Rate limiting caps action rates such as firing or API calls to deter automated exploits. It protects servers from floods and keeps gameplay within fair human bounds.

Synonyms

throttling

  • API Abuse Prevention
