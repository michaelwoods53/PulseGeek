Verifiable Random Function

A verifiable random function produces randomness tied to a public key, along with a proof that anyone can verify. It supports fair leader election and lotteries without trusted parties.

In detail

A verifiable random function is like a digital signature that outputs a pseudorandom value plus a proof the value is correct for a given input and public key. Anyone can verify the proof, ensuring the random output was not biased or tampered with. Blockchains use VRFs to select validators, assign duties, or run lotteries in a way that is unpredictable yet publicly checkable. VRFs help prevent manipulation, reduce grinding attacks, and improve fairness. They are often combined with entropy beacons or additional randomness sources to strengthen security in adversarial environments, balancing unpredictability with transparency.

Synonyms
