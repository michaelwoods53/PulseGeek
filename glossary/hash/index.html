Hash

A calculated value used to verify file integrity.

In detail

Hashes such as MD5 or SHA-1 are computed values that help confirm a BIOS or game dump matches a known good copy. They aid in auditing and preservation.

Synonyms

checksum, digest

Related terms

  • MD5
  • SHA-1
