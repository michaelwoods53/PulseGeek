Pole Spacing

The distance between lighting poles along a roadway.

In detail

Pole spacing is set by mounting height, roadway width, and photometric distribution. In smart designs it also considers communication coverage and sensor placement.

Synonyms

pole distance, luminaire spacing

Related terms

  • Photometric Distribution
  • Smart Pole
  • Network Coverage
