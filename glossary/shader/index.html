Shader

GPU code that controls how vertices and pixels are processed.

In detail

Shaders are small programs executed on the GPU to transform vertices, compute lighting, and apply effects. Common types include vertex, fragment/pixel, geometry, and compute shaders, and they can be authored by code or graph tools.

Synonyms

shader program, material shader
