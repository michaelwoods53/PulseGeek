Common Grid Model Exchange Specification

A set of IEC CIM profiles used to exchange consistent transmission and distribution network models across organizations, supporting planning, operations, and market processes with standardized semantics and validation rules.

In detail

The Common Grid Model Exchange Specification is a standardized set of profiles built on the IEC Common Information Model that enables interoperable exchange of power system network models. Widely used in Europe under ENTSO-E governance, CGMES defines how to represent topology, equipment parameters, steady-state hypotheses, and state variables across multiple data sets so that parties can build a common grid model. By aligning semantics and providing validation rules, CGMES reduces translation work and errors when sharing models between transmission system operators, distribution utilities, and market platforms. It supports planning studies, congestion analysis, security assessment, and cross-border coordination. While focused on network model exchange, CGMES can complement operational systems by providing accurate, versioned models for ADMS and DERMS integration. Utilities benefit from traceable model lineage, better consistency with field assets, and a scalable path to multi-party coordination.

Synonyms
