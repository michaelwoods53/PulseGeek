Latency

Time it takes for input to travel to the server and back.

In detail

Latency is the round trip time between a player’s input and the visible response on screen. It includes controller, display, network, server processing, and encoding delays.

Synonyms

delay, lag

Abbreviations

RTT

Related terms

  • Input Latency
  • Jitter
