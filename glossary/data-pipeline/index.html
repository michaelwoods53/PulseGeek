Data Pipeline

Systems that move, clean, and prepare data from sources into models and dashboards on a reliable schedule.

In detail

A data pipeline connects source systems, transforms raw data, and loads it into destinations for analytics and AI. In finance, sources can include ERP, banking feeds, and procurement tools. Good pipelines handle validation, schema changes, late arrivals, and logging. Reliability and lineage are critical for auditability. Well-designed pipelines reduce data drift, improve model accuracy, and support timely dashboards for decision-making.

Synonyms

data integration pipeline, etl pipeline

Abbreviations

ETL, ELT

  • Model Drift
  • Data Drift
