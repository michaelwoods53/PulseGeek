Compression

Reducing audio data size with perceptual or adaptive codecs.

In detail

Game audio uses codecs like Vorbis or ADPCM to reduce memory and disk usage. Choosing formats depends on platform constraints, latency, and artifact tolerance.

Synonyms

codec compression
